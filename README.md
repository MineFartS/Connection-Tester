# Connection Tester

### Easily test if a hostname is reachable

## Usage:

Copy and paste the following line into powershell:

    irm https://raw.githubusercontent.com/MineFartS/Connection-Tester/refs/heads/main/run.ps1 | iex

## Example:

    Hostname: philh.myftp.biz

```
philh.myftp.biz - Online

Waiting for 3 seconds, press a key to continue ...
```
